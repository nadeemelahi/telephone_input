Description

This script will format an input of type="text" or type="tel" into XXX-XXX-XXXX.

It listens on events click and keyup allowing only digits, arrow keys and delete and backspace.

This script works on all modern browsers including IE9 and above.

Usage

Instantiate an instance of Tel_inp_formatter(my_input_id) passing in the id of your telephone input.

For example, if you have an input with an id="my_tel_input" then you would code as follows:

var tel_inp_formatter
= new Tel_inp_formatter(
"my_tel_input"
);

Once a valid telephone number if fully entered into the tel input the blur() method is called; taking away focus.

Optionally, pass the id of the next input in a form, and that input will get focus after the tel input is filled with a valid telephone number.

Such as follows:

var tel_inp_formatter
= new Tel_inp_formatter(
"my_tel_input",
"my_next_input"
);

Advanced Notes

The Tel_inp_formatter class uses the .addEventListener() method which is not supported by IE8 and below. There are two click handlers and a keyup handler. The names of functions passed to .addEventListener() methods are prepended with TIF_ for namespace safety. The minified version, minified using google's closure compiler, changes the click event handler function names for to 'n' and 'q'. So you must ensure any other code you use does not trample on those names. The keyup event handler get's inlined(an anonymous function) -so no namespace trampling concerns there.
